# Transit PH - Naga City

A modern, interactive transit navigation application tailored for Naga City. This app helps users find the best routes and transportation modes (Jeepney, Tricycle, E-Jeep) between local destinations.

## ✨ Features

- **Interactive Map**: Full-screen map interface for visual navigation.
- **Smart Route Search**: Find routes between popular Naga City landmarks.
- **Multi-Modal Transport**: Select between different transit types to see relevant options.
- **Modern UI**: Built with a mobile-first approach using Tailwind CSS 4 and glassmorphic design elements.
- **Fast & Responsive**: Powered by Next.js for optimal performance.

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/transit-ph.git
   cd transit-ph
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management**: React Hooks (useState, useEffect)
- **Icons/UI**: Custom CSS & SVGs

## 📁 Project Structure

- `src/app`: Main application logic and pages.
- `src/components`: Reusable UI components.
- `public`: Static assets (icons, images).
